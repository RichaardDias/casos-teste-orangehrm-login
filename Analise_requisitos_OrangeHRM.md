
# 📋 Análise de Requisitos – OrangeHRM  
**Base:** Plano de Testes Funcionais  
**Objetivo:** Validar se os requisitos funcionais estão claramente definidos e testáveis.

---

## ✅ 1. Login na Plataforma

**Requisitos Identificados:**
- Deve permitir login com credenciais válidas.
- Deve rejeitar login com senha errada.
- Deve exibir alertas se campos estiverem vazios.

**Status da Análise:**  
✔️ Requisitos claros e bem testados.
